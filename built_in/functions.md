## Built-In Functions

### abs()
Return the absolute value of an integer, floating-point, or an object implementing ` __abs__() `. If the argument is a complex number, its magnitude is returned.

**Parameters**:
- **x** - variable to get the absolute value of.

```py
>>> number: int = 123
>>> abs(number)
123
>>> number:int = -123
>>> abs(number)
123
```
